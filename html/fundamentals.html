🎯 Konsep Kunci

📦 Variables (Variabel)

Teori: Variable adalah tempat penyimpanan data dalam program. Seperti kotak yang diberi label, setiap kotak (variable) menyimpan satu nilai.

Analogi Kehidupan Nyata:

  • 🏠 Rumah → Program
  • 📦 Kotak dengan label → Variable
  • 🎁 Barang di dalam kotak → Nilai (value)

Contoh: Kotak bernama "nama" menyimpan "Ari". Kotak bernama "umur" menyimpan angka 25.

nama = "Ari"      # Kotak 'nama' berisi "Ari"
umur = 25         # Kotak 'umur' berisi 25
tinggi = 1.75     # Kotak 'tinggi' berisi 1.75

🏷️ Data Types (Tipe Data)

Teori: Setiap data memiliki tipe. Tipe menentukan apa yang bisa dilakukan dengan data tersebut.

Analogi Kehidupan Nyata:

  • 📝 String = Teks (seperti surat atau buku). Contoh: "Ari", "Hello"
  • 🔢 Integer = Angka bulat (seperti penghitung barang). Contoh: 25, 100
  • 🟠 Float = Angka desimal (seperti tinggi badan atau berat). Contoh: 1.75, 3.14
  • ✅/❌ Boolean = Benar atau Salah (seperti ya/tidak). Contoh: True, False
str_var = "Hello"     # String (teks)
int_var = 42          # Integer (angka bulat)
float_var = 3.14      # Float (angka desimal)
bool_var = True       # Boolean (benar/salah)

# Cek tipe data menggunakan type()
print(type(str_var))    # 
print(type(int_var))    #

➕ Operators (Operator)

Teori: Operator adalah simbol yang digunakan untuk melakukan operasi pada data.

Analogi Kehidupan Nyata:

  • ➕ Penjumlahan (+) = Menambah uang di dompet
  • ➖ Pengurangan (-) = Mengeluarkan uang dari dompet
  • ✖️ Perkalian (*) = Menggandakan barang
  • ➗ Pembagian (/) = Membagi kue ke beberapa orang
a = 10
b = 3

# Arithmetic Operators
print(a + b)  # 13 (tambah)
print(a - b)  # 7  (kurang)
print(a * b)  # 30 (kali)
print(a / b)  # 3.333... (bagi)
print(a ** 2) # 100 (pangkat)

💬 Input & Output (Masukan & Keluaran)

Teori: Program perlu berkomunikasi dengan user. Keluaran (output) menampilkan hasil, masukan (input) menerima data dari user.

Analogi Kehidupan Nyata:

  • 📢 print() = Penjual menyuarakan harga
  • 👂 input() = Pembeli menjawab pertanyaan penjual
# Output - menampilkan informasi
print("Hello, World!")

# Input - menerima data dari user
nama = input("Siapa nama Anda? ")

# Menggabungkan input dan output
print(f"Halo, {nama}!")  # f-string untuk formatting

🔄 Type Conversion (Konversi Tipe)

Teori: Kadang kita perlu mengubah satu tipe data menjadi tipe lain. Proses ini disebut type conversion atau casting.

Analogi Kehidupan Nyata:

  • 💵 Mengubah uang rupiah menjadi dolar (dari int ke float)
  • 📝 Menulis angka di kertas (dari int ke string)
# String ke Integer
nilai_text = "25"
nilai_angka = int(nilai_text)
print(nilai_angka + 5)  # 30 (berhasil karena sudah integer)

# Integer ke String
umur = 20
text = str(umur)
print("Umur saya: " + text)  # Umur saya: 20
