1. Variabel & Tipe Data

Variabel adalah tempat menyimpan data. Setiap data memiliki tipe yang berbeda.

📝 String (Teks)

Teks atau karakter

nama = "Ari"

🔢 Integer (Bulat)

Angka tanpa koma

umur = 17

🟠 Float (Desimal)

Angka dengan koma

tinggi = 1.75

✅ Boolean

Benar atau Salah

aktif = True

Contoh Kode:

variabel.py 
# Membuat variabel
nama = "Ari"           # String
umur = 17              # Integer
tinggi = 1.75          # Float
aktif = True           # Boolean

# Menampilkan data
print(f"Nama: {nama}")
print(f"Umur: {umur} tahun")
print(f"Tinggi: {tinggi} m")

# Cek tipe data
print(type(nama))   # <class 'str'>
➕

2. Operator

Operator digunakan untuk melakukan operasi pada data.

+ Tambah - Kurang * Kali / Bagi // Bagi Bulat % Sisa Bagi ** Pangkat

Contoh Kode:

operator.py 
a = 10
b = 3

# Arithmetic
print(a + b)   # 13 (tambah)
print(a - b)   # 7  (kurang)
print(a * b)   # 30 (kali)
print(a / b)   # 3.33 (bagi)
print(a // b)  # 3 (bagi bulat)
print(a % b)   # 1 (sisa bagi)
print(a ** b)  # 1000 (pangkat)
💬

3. Input & Output

Input untuk menerima data dari user, output untuk menampilkan data.

Contoh Kode:

io.py 
# Output - Menampilkan data
print("Halo Dunia!")
print(f"Umur: {17}")

# Input - Menerima data dari user
nama = input("Siapa namamu? ")
print(f"Halo, {nama}!")

# Input angka
umur = int(input("Berapa umurm? "))
print(f"5 tahun lagi: {umur + 5}")
🔄

4. Type Conversion

Mengubah satu tipe data ke tipe lain.

Contoh Kode:

conversion.py 
# String ke Integer
nilai = "25"
angka = int(nilai)      # 25
print(angka + 5)        # 30

# Integer ke String
umur = 17
text = str(umur)       # "17"
print("Umur: " + text)

# String ke Float
harga = "15000.50"
harga_float = float(harga)  # 15000.5
