⌨️ Quick Syntax Reference

Tipe Data Dasar

# String, Integer, Float, Boolean
text = "Hello"
number = 42
decimal = 3.14
flag = True

# List, Tuple, Dictionary, Set
list_data = [1, 2, 3]
tuple_data = (1, 2, 3)
dict_data = {"key": "value"}
set_data = {1, 2, 3}

Control Flow

# If Statement
if condition:
    do_something()
elif other_condition:
    do_other()
else:
    do_default()

# For Loop
for i in range(5):
    print(i)

# While Loop
while condition:
    do_something()

Functions

# Function Definition
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

# Function Call
result = greet("Ari")
print(result)

# Lambda
square = lambda x: x ** 2
print(square(5))

Working with Files

# Reading
with open("file.txt", "r") as f:
    content = f.read()

# Writing
with open("file.txt", "w") as f:
    f.write("Hello")

# Appending
with open("file.txt", "a") as f:
    f.write("More")

Classes

class MyClass:
    def __init__(self, name):
        self.name = name
    
    def greet(self):
        return f"Hello, {self.name}!"

obj = MyClass("Ari")
print(obj.greet())

Error Handling

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
except Exception as e:
    print(f"Error: {e}")
finally:
    print("Cleanup done")

💡 Tips & Best Practices

📝 Naming Convention

Gunakan snake_case untuk variables dan functions, PascalCase untuk classes

💬 Documentation

Tulis docstrings untuk functions dan classes agar mudah dipahami

🧪 Testing

Buat unit tests untuk memastikan code berfungsi dengan baik

🔍 Debugging

Gunakan print statements atau debugger untuk track errors
