🎯 PILIHAN MEMBUKA:

Option 1 - MUDAH (Double Click)
   - Cari file: index.html di folder html/
   - Double-click untuk buka di browser default
   - Naviguasi dari halaman utama

Option 2 - RECOMMENDED (Python Server)
   - Buka terminal/command prompt
   - Ketik: cd html
   - Ketik: python3 -m http.server 8000
   - Buka browser: http://localhost:8000

Option 3 - Exercise Menu (Interaktif)
   - Buka terminal/command prompt
   - Ketik: cd html/exercises
   - Ketik: python3 run_exercises.py
   - Pilih exercise dari menu
