🎯 Konsep Kunci

🔁 For Loop (Perulangan Untuk)

Teori: For loop digunakan untuk mengulangi blok kode beberapa kali, biasanya untuk mengakses setiap item dalam daftar.

Analogi Kehidupan Nyata:

  • 📚 Membaca setiap halaman buku satu per satu
  • 🎓 Memangil daftar siswa untuk presensi
  • 🏪 Kasir memproses setiap barang belanja di keranjang
# Perulangan dengan range (0, 1, 2, 3, 4)
for i in range(5):
    print(i)

# Perulangan untuk setiap item dalam list
buah = ["apel", "jeruk", "mangga"]
for item in buah:
    print(item)

# Analogi: Untuk setiap halaman dalam buku, baca isinya
# for halaman in buku:
#     baca_halaman(halaman)

⏰ While Loop (Perulangan Selama)

Teori: While loop mengulangi kode selama kondisi masih benar (True). Berbeda dengan for loop yang tahu jumlah perulangan.

Analogi Kehidupan Nyata:

  • 🚗 Berkendara selama belum sampai tujuan
  • 💤 Tidur selama masih mengantuk
  • 🍽️ Makan selama masih lapar
# Perulangan selama kondisi benar
count = 0
while count < 5:
    print(count)
    count += 1

# Analogi: "Selama uang masih ada, terus belanja"
uang = 100
while uang > 0:
    print(f"Sisa uang: {uang}")
    uang -= 10

🚦 If/Else (Pengambilan Keputusan)

Teori: If/else digunakan untuk membuat program membuat keputusan berdasarkan kondisi. Mirip dengan traffic light yang menentukan apakah boleh jalan atau berhenti.

Analogi Kehidupan Nyata:

  • 🌧️ Jika hujan, bawa payung. Jika tidak, tidak perlu.
  • 🔴🟡🟢 Traffic light: merah = berhenti, hijau = jalan
  • 🎓 Jika nilai ≥ 70 = lulus, jika tidak = tidak lulus
nilai = 85
if nilai >= 90:
    print("Grade: A")
elif nilai >= 80:
    print("Grade: B")
else:
    print("Grade: C")

# Analogi sehari-hari
cuaca = "hujan"
if cuaca == "hujan":
    print("Bawa payung")
else:
    print("Tidak perlu payung")

🔧 Functions (Fungsi)

Teori: Function adalah blok kode yang dapat digunakan kembali. Membantu menghindari duplikasi kode dan membuat program lebih terorganisir.

Analogi Kehidupan Nyata:

  • 🍕 Resep masak = Template (function)
  • 👨‍⚕️ Dokter = Menerima input (gejala) → Memberikan output (resep)
  • 🔌 Steker listrik = Dapat digunakan di mana saja berkali-kali
def hitung_luas_persegi(sisi):
    """Function dengan 1 parameter"""
    luas = sisi * sisi
    return luas

def salam(nama, waktu="pagi"):
    """Function dengan parameter default"""
    return f"Selamat {waktu}, {nama}!"

# Menggunakan function
print(hitung_luas_persegi(5))      # 25
print(salam("Ari"))                # Selamat pagi, Ari!
print(salam("Budi", "sore"))       # Selamat sore, Budi!

📄 File I/O (Baca & Tulis File)

Teori: Program perlu menyimpan data secara permanen. Dengan file I/O, kita bisa menulis data ke file dan membacanya kembali nanti.

Analogi Kehidupan Nyata:

  • 📝 Menulis buku (write) = Menyimpan data ke file
  • 📖 Membaca buku (read) = Membaca data dari file
  • 📚 Pustaka = Penyimpanan file di hard disk
# Menulis ke file
data = ["Nama: Ari\n", "Umur: 20\n", "Kota: Bandung\n"]
with open("data.txt", "w") as f:
    f.writelines(data)

# Membaca dari file
with open("data.txt", "r") as f:
    content = f.read()
    print(content)

🛡️ Exception Handling (Penanganan Kesalahan)

Teori: Program pasti pernah mengalami error. Exception handling membantu program menangani error dengan graceful, tidak crash.

Analogi Kehidupan Nyata:

  • 🏥 Dokter siap mengobati pasien yang sakit (error)
  • 🎯 Airbag mobil = Siap jika ada kecelakaan (error)
  • 🛟 Sabuk pengaman = Proteksi saat error terjadi
try:
    # Kode yang mungkin error
    file = open("file_yang_tidak_ada.txt")
    content = file.read()
except FileNotFoundError:
    # Apa yang dilakukan jika error terjadi
    print("Maaf, file tidak ditemukan!")
finally:
    # Kode ini SELALU dijalankan (opsional)
    print("Selesai")

#!/usr/bin/env python3
"""
Intermediate - Exercise 3: File Operations & Exceptions
========================================================

SOAL:
1. Buat file dan tulis data ke file
2. Baca file dan tampilkan isi
3. Lakukan error handling saat membaca file
4. Append data ke file existing
5. Hitung jumlah baris di file
"""

file_path = "data.txt"
data = [
    "Nama: Ari\n",
    "Umur: 20\n",
    "Kota: Bandung\n",
]

# Tulis ke file
try:
    with open(file_path, "w") as f:
        f.writelines(data)
    print(f"✓ File '{file_path}' berhasil dibuat")
except IOError as e:
    print(f"✗ Error: {e}")
