1. Loops (Perulangan)

For loop dan while loop untuk mengulang kode secara efisien.

for i in range() while break/continue nested loops

Contoh Kode:

loops.py 
# For loop -ulang 5 kali
for i in range(5):
    print(f"Perulangan ke-{i}")

# While loop -ulang selama kondisi benar
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

# List comprehension
kuadrat = [x**2 for x in range(5)]
🚦

2. Conditions (Pengambilan Keputusan)

If, elif, else untuk membuat program memutuskan sesuatu.

if elif else operator perbandingan and/or/not

Contoh Kode:

conditions.py 
nilai = 85

# If-Else sederhana
if nilai >= 90:
    print("Grade: A")
elif nilai >= 80:
    print("Grade: B")
elif nilai >= 70:
    print("Grade: C")
else:
    print("Grade: D")

# Dengan operator logika
usia = 17
if usia >= 18 and usia <= 25:
    print("Dewasa muda")
⚙️

3. Functions (Fungsi)

Blok kode yang bisa digunakan ulang berkali-kali.

def parameter return *args lambda

Contoh Kode:

functions.py 
# Function dasar
def sapa(nama):
    return f"Halo, {nama}!"

# Function dengan default parameter
def greet(nama, waktu="pagi"):
    return f"Selamat {waktu}, {nama}!"

# Lambda function
kuadrat = lambda x: x ** 2

# Function dengan multiple return
def kalkulasi(a, b):
    return a+b, a-b, a*b

print(sapa("Ari"))          # Halo, Ari!
print(kuadrat(5))          # 25
print(kalkulasi(10, 3))     # (13, 7, 30)
📝

4. String Operations

Memotong, mengubah, dan memanipulasi teks.

.upper()/.lower() .split()/.join() .replace() f-string

Contoh Kode:

strings.py 
text = "Hello World"

# Mengubah case
print(text.upper())     # HELLO WORLD
print(text.lower())     # hello world

# Memotong
print(text.split())    # ['Hello', 'World']

# Replace
print(text.replace("World", "Python"))  # Hello Python

# F-string (paling penting!)
nama = "Ari"
umur = 17
print(f"Nama: {nama}, Umur: {umur}")
📂

5. File Handling

Membaca dan menulis data ke file.

open() read() write() with

Contoh Kode:

file_io.py 
# Menulis ke file
with open("data.txt", "w") as f:
    f.write("Halo dunia!\n")
    f.write("Ini baris kedua")

# Membaca file
with open("data.txt", "r") as f:
    content = f.read()
    print(content)

# Menambah data (append)
with open("data.txt", "a") as f:
    f.write("\nBaris baru")
🛡️

6. Exception Handling

Menangani error dengan elegan agar program tidak crash.

try except finally raise

Contoh Kode:

exception.py 
try:
    hasil = 10 / 0
except ZeroDivisionError:
    print("Tidak boleh bagi nol!")
except Exception as e:
    print(f"Error: {e}")
finally:
    print("Selalu dijalankan")

# Contoh lain
try:
    angka = int(input("Masukkan angka: "))
except ValueError:
    print("Input harus angka!")
