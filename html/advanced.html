1. Classes & Objects

Class adalah blueprint, object adalah implementasi nyata.

class __init__ self attributes methods

Contoh Kode:

class.py 
# Membuat class Siswa
class Siswa:
    def __init__(self, nama, umur):
        self.nama = nama    # attribute
        self.umur = umur   # attribute
    
    def sapa(self):        # method
        return f"Halo, {self.nama}!"

# Membuat object (instance)
siswa1 = Siswa("Ari", 17)
siswa2 = Siswa("Budi", 18)

print(siswa1.sapa())  # Halo, Ari!
🧬

2. Inheritance (Pewarisan)

Class anak mewarisi semua attribute dan method dari class parent.

class Parent class Child(Parent) super()

Contoh Kode:

inheritance.py 
# Parent class
class Manusia:
    def __init__(self, nama):
        self.nama = nama
    
    def makan(self):
        print(f"{self.nama} makan...")

# Child class
class Siswa(Manusia):
    def __init__(self, nama, kelas):
        super().__init__(nama)  # panggil __init__ parent
        self.kelas = kelas
    
    def belajar(self):
        print(f"{self.nama} belajar...")

# Usage
siswa = Siswa("Ari", "XII-IPA1")
siswa.makan()   # dari parent
siswa.belajar() # dari child
🔄

3. Polymorphism

Method dengan nama sama tapi perilaku berbeda.

method overriding duck typing abstract

Contoh Kode:

polymorphism.py 
class Kucing:
    def suara(self):
        return "Meong"

class Anjing:
    def suara(self):
        return "Guk-guk"

# Polymorphism - method sama, hasil berbeda
def cetak_suara(hewan):
    print(hewan.suara())

kucing = Kucing()
anjing = Anjing()

cetak_suara(kucing)  # Meong
cetak_suara(anjing)  # Guk-guk
🔒

4. Encapsulation

Menyembunyikan data dengan akses terbatas.

private (__attr) getter setter @property

Contoh Kode:

encapsulation.py 
class AkunBank:
    def __init__(self, saldo):
        self.__saldo = saldo  # private (underscore)
    
    @property
    def saldo(self):
        return self.__saldo
    
    @saldo.setter
    def saldo(self, nilai):
        if nilai >= 0:
            self.__saldo = nilai
    
    def setor(self, jumlah):
        self.__saldo += jumlah
    
    def tarik(self, jumlah):
        if jumlah <= self.__saldo:
            self.__saldo -= jumlah

# Usage
akun = AkunBank(1000000)
print(akun.saldo)  # 1000000
akun.setor(500000)
print(akun.saldo)  # 1500000
✨

5. Special Methods

Method khusus dengan double underscore __method__.

__init__ __str__ __repr__ __add__

Contoh Kode:

special.py 
class Bilangan:
    def __init__(self, nilai):
        self.nilai = nilai
    
    def __str__(self):
        return f"Bilangan({self.nilai})"
    
    def __repr__(self):
        return f"Bilangan(nilai={self.nilai})"
    
    def __add__(self, lain):
        return self.nilai + lain.nilai

a = Bilangan(10)
b = Bilangan(5)

print(a)        # Bilangan(10)
print(a + b)    # 15
