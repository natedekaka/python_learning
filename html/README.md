## 🎯 Cara Menggunakan

### Untuk Belajar dari Browser:
1. Buka `index.html` di browser
2. Navigasi ke level yang sesuai (Fundamentals → Intermediate → Advanced)
3. Baca teori dan konsep kunci
4. Klik tombol "Download" untuk exercises
5. Lihat References untuk informasi lebih detail

### Untuk Mengerjakan Exercises:
1. Download file exercise
2. Buka dengan text editor atau IDE favorit (VS Code, PyCharm, Sublime, dll)
3. Ikuti instruksi di comments
4. Jalankan file dengan: `python3 filename.py`
5. Eksperimen dan modifikasi code
6. Referensi ke halaman HTML jika butuh bantuan

### Menggunakan Exercise Menu:
```bash
cd html/exercises
python3 run_exercises.py
```
- Pilih exercise dari menu
- Program akan menjalankan file exercise
- Bisa juga view code atau list semua exercises

## 📞 Troubleshooting

### Browser tidak bisa buka file
- Gunakan Python server: `python3 -m http.server 8000`
- Atau gunakan browser: `right-click > Open with > Browser`

### Exercise tidak berjalan
- Pastikan Python 3.x sudah terinstall
- Jalankan dari terminal: `python3 filename.py`
- Baca error message dengan teliti

### File exercise tidak ketemu
- Pastikan Anda di folder `html/exercises/`
- Download file terlebih dahulu dari halaman Exercises
- Atau copy dari folder `references/` jika tersedia
