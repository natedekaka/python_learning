💡 TIPS BELAJAR YANG EFEKTIF:
═══════════════════════════════════════════════════════════════════

1. 🎯 FOKUS PADA KONSEP, BUKAN SYNTAX
   • Pahami MENGAPA code berfungsi
   • Jangan hanya hafal syntax
   • Tanyakan kepada diri sendiri: "Apa yang terjadi di sini?"

2. 💻 PRAKTIK DENGAN HANDS-ON
   • Ketik semua kode sendiri (jangan copy-paste)
   • Ubah nilai-nilai dan lihat apa yang berubah
   • Buat error sendiri untuk belajar

3. 🐛 DEBUGGING ADALAH SKILL PENTING
   • Gunakan print() untuk debug
   • Baca error message dengan teliti
   • Google error yang tidak dimengerti

4. 📚 ULANG & KONSOLIDASI
   • Ulang materi setelah beberapa hari
   • Buat catatan pribadi
   • Jelaskan kepada orang lain

5. 🚀 BUILD PROJECTS KECIL
   • Jangan hanya teori
   • Buat aplikasi sederhana (todo list, calculator, dll)
   • Challenge diri sendiri

🔍 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════

Q: File tidak bisa dijalankan?
A: Pastikan Python 3 sudah terinstall
   $ python3 --version
   $ python3 nama_file.py

Q: Error "Permission denied"?
A: Jalankan dengan python3 di depan
   $ python3 file.py    (BENAR)
   $ file.py            (SALAH)

Q: Meminta input tapi tidak tahu apa?
A: Tekan Enter atau ketik nomor (sesuai instruksi)
   File 05_files_exceptions.py meminta input untuk demo

Q: Tidak mengerti output program?
A: Baca komentar di file Python
   Semua file punya penjelasan detail
