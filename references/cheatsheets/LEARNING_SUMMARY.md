## Common Operators Reference

### Arithmetic
| Operator | Meaning | Example |
|----------|---------|---------|
| + | Addition | 5 + 3 = 8 |
| - | Subtraction | 5 - 3 = 2 |
| * | Multiplication | 5 * 3 = 15 |
| / | Division | 6 / 3 = 2.0 |
| // | Floor Division | 7 // 2 = 3 |
| % | Modulo (Remainder) | 7 % 2 = 1 |
| ** | Power | 2 ** 3 = 8 |

### Comparison
| Operator | Meaning |
|----------|---------|
| == | Equal to |
| != | Not equal to |
| < | Less than |
| > | Greater than |
| <= | Less than or equal |
| >= | Greater than or equal |

### Logical
| Operator | Meaning |
|----------|---------|
| and | Both conditions True |
| or | At least one True |
| not | Negates condition |

## Quick Tips
1. **Variables**: `name = "value"` (no type declaration needed)
2. **Lists**: `lst = [1, 2, 3]` - mutable, ordered
3. **Dicts**: `dict = {key: value}` - key-value pairs
4. **Loops**: `for item in collection:` or `while condition:`
5. **Functions**: `def function_name(params):` with `return` statement
